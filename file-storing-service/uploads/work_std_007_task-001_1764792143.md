# Инструкция по запуску
## Требования
- Go 1.21 или выше

## Установка и запуск

### 1. Скачать зависимости

```
go mod download
```

Установить 2 основных пакета, используемых при разработке и нужных для корректной работы:
- `gopkg.in/yaml.v3` - парсинг YAML файлов
- `go.uber.org/dig` - Dependency Injection контейнер

### 2. Запустить приложение
**Способ 1**

Открыть терминал в папке HSE_Bank_App и ввести:
```
go run ./cmd
```

**Способ 2**

Открыть терминал в папке cmd и ввести:
```
go run .
```

**Способ 3**

Открыть код в любой подходящей IDE, например GoLand и запустить по-нормальному, предварительно
всё так же установив зависимости и скачав все необходимые пакеты
